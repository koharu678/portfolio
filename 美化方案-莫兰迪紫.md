## 🎨 莫兰迪紫 · Apple 风格美化方案

---

### 一、设计关键词

`低饱和度` `磨砂玻璃` `圆润柔和` `大量留白` `微妙的层级阴影`

整体感觉像 iOS 设置页面或 Mac 系统偏好设置——干净、安静、不争不抢。

### 三、视觉变化一览

**字级与排版**

- 字体改用系统原生栈：`-apple-system, 'SF Pro Display', 'PingFang SC'`（苹果味来源）
- 标题字重降低半档（`font-semibold` → `font-medium`），更轻盈
- 行距加大（`leading-relaxed` → `leading-loose`），呼吸感更强

**卡片与容器**

- 卡片背景：半透明毛玻璃 `bg-white/60 backdrop-blur-md`
- 圆角全面加大：`rounded-2xl` → `rounded-3xl`
- 内边距加大：`p-5` → `p-7`，更松弛
- 阴影从锐利变柔和：`shadow-sm` → `shadow-[0_2px_16px_rgba(0,0,0,0.04)]`
- hover 阴影也变柔和，不跳太高

**圆角与圆润感**

- 标签、按钮、图标容器全部 `rounded-full`（胶囊/圆形）
- 项目卡片 `rounded-3xl`，内部视频区也带圆角

**莫兰迪色标签**

- 之前：🔵蓝 `bg-blue-50 text-blue-700`、🟣紫 `bg-purple-50 text-purple-700`
- 现在：统一压灰——`bg-[#EDE8F0] text-[#7B7B9C]`、`bg-[#EDF0EB] text-[#8FA88F]`

**导航栏**

- 毛玻璃感加强：`bg-white/40 backdrop-blur-xl`
- 去掉底部边框，完全透明悬浮感

### 四、调整范围

| 层级 | 改动 |
|------|------|
| 色板、字体、圆角、阴影 | **全面替换** |
| 卡片背景（毛玻璃） | **重写** |
| 标签配色 | **全部改为莫兰迪低饱和版** |
| Hero 渐变文字 | 蓝→蓝紫 改为 灰紫→暖灰 渐变 |
| 布局骨架 | **不变**（不改 grid、flex、max-w） |
